🛍️ Easy Shopping Via Mobile

Este projeto foi desenvolvido como parte dos meus estudos em HTML e CSS, com foco em evolução prática e construção de layouts modernos e responsivos.

📸 Preview

preview.png


💻 Sobre o projeto

A proposta foi criar uma landing page moderna simulando um aplicativo de compras mobile, com divisão de layout entre texto e imagem.

Durante o desenvolvimento, utilizei:

HTML para estruturação da página
CSS para estilização
Flexbox para organização do layout
Media queries para responsividade
📱 Responsividade

O projeto foi adaptado para diferentes tamanhos de tela, garantindo uma boa experiência em:

💻 Desktop
📱 Tablet
📲 Mobile
🚧 Desafios

Durante o desenvolvimento deste projeto, enfrentei algumas dificuldades importantes, principalmente em:

Alinhamento de elementos na tela
Uso correto do Flexbox
Posicionamento da imagem e do texto
Ajustes de responsividade

Esses desafios fizeram parte do processo e foram essenciais para o meu aprendizado.

🎯 Aprendizado

Esse projeto contribuiu muito para a minha evolução como desenvolvedora.

Com ele, consegui:

Melhorar a organização do meu código
Entender melhor o funcionamento do Flexbox
Resolver problemas de layout na prática
Trabalhar com responsividade de forma mais eficiente

Além disso, consegui adaptar o projeto e deixá-lo do jeito que eu quis, aplicando meu próprio estilo e entendimento.
